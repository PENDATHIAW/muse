# MUSE

MUSE est un site catalogue/e-commerce pour des objets personnalisables conçus en impression 3D.

## Objectif

Créer une vitrine premium permettant de présenter les objets par univers, avec une interface administrateur pour gérer :

- les univers de produits ;
- les objets du catalogue ;
- les visuels ;
- les descriptions ;
- les prix ;
- les demandes clients.

## Univers prévus

- MUSE Kitchen
- MUSE Tech / Charge Guard
- MUSE Wedding
- MUSE Maison
- MUSE Business
- MUSE Femme
- MUSE Décoration

## Stack envisagée

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- Interface admin sécurisée
