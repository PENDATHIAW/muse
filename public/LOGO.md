# Logo MUSE

Placez votre logo officiel ici :

**`public/logo-muse.png`**

## Votre identité

- Symbole **M** stylisé
- Texte **MUSE** (capitales, espacement large)
- Signature : **Shaping your ideas**

Tant que le fichier n’est pas présent, le site affiche un fallback texte avec la signature.

## Conseils d’export

1. Exportez en PNG haute résolution depuis votre fichier source
2. Pour l’accueil : le visuel avec fond sombre fonctionne très bien (cadre intégré au site)
3. Pour le header : une version plus compacte ou fond transparent est idéale

## Remplacement

Remplacez simplement le fichier `logo-muse.png` — aucune modification de code nécessaire.
