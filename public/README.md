# Dossier public — vos fichiers visuels MUSE

Tous les fichiers ici sont servis directement par le site.

## Où ajouter quoi

| Emplacement | Fichier à ajouter |
|-------------|-------------------|
| `logo-muse.png` | Votre logo (à la racine de ce dossier) |
| `catalogue_muse_2026.pdf` | Votre catalogue PDF (à la racine) |
| `products/[slug]/main.jpg` | Photo principale de chaque produit |
| `univers/[slug]/cover.jpg` | Image de couverture de chaque univers |

## Après avoir ajouté une image

1. Push sur GitHub
2. Admin → Produit → Images → chemin : `/products/[slug]/main.jpg`
3. Le site affiche l'image automatiquement
